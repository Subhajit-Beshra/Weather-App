# Weather-App
“Weather App is a clean and minimal web application that lets you instantly look up real-time weather information for any city around the world. Simply type in a city name, hit search, and get key weather details like temperature, humidity, wind speed, and current conditions — all in one glance.”

Key Features:
🔍 City Search — Look up weather for any city worldwide
🌡️ Temperature — Displayed in Celsius for easy reading
💧 Humidity — Get real-time moisture levels in the air
⛅ Weather Condition — Understand the current sky condition instantly

Built using HTML, CSS, and JavaScript, with real-time API integration, and deployed seamlessly on GitHub and Netlify.
