# Weather-App
Weather App is a clean and minimal web application that lets you instantly look up real-time weather information for any city around the world. Simply type in a city name, hit search, and get the current temperature, wind speed, and weather condition — all in one glance.
Key Features:

🔍 City Search — Look up weather for any city worldwide
🌡️ Temperature — Displayed in Celsius for easy reading
💨 Wind Speed — Real-time wind data at your fingertips
⛅ Weather Condition — Know exactly what the sky looks like, wherever you are.

Built with HTML, CSS, and JavaScript, and deployed seamlessly on GitHub & Netlify.
